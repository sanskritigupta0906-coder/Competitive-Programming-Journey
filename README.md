# Competitive-Programming-Journey
This repository contains my solutions to various Codeforces problems written in C++.

The purpose of this repository is not just to store accepted solutions, but also to document my thought process while solving problems. Many solutions include detailed comments explaining the logic so that beginners (including my future self 😄) can understand the approach easily.

Repository Structure

Each problem has its own folder containing:

solution.cpp – Accepted solution with comments
README.md (optional) – Notes, observations, and approach
